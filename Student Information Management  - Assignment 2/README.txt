Valid Input for Student Records Should be as follows:
"Name","Roll Number","Marks"
Ex:
John,1,45,67,77
Matthew,2,89,43,56

1. View All Records
Displays all records in StudentRecords.txt
2. Edit Records
Edits an existing record
3.Delete record
Removes a record from txt file
4. View by roll number
Takes a roll number as input and displays record
5. Save 
Must be ran before exit to update all changes to txt file
6. Exit
Ends the program
